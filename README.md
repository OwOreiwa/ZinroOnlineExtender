人狼Onlineえくすてんだー
====
人狼Online（ https://zinro.net/ ）が使いにくいので自分で機能を追加する試みです  
現在少ない機能しかないですが他にも追加する予定です

DEMO
----

Usage
----
* 雑談系/身内/合言葉の村がグレーでローライトされます
* ブラウザアクションボタン押下で雑談系/身内/合言葉の村が非表示になります  
（精度は悪いので複数回実行してください）
* CPU/観戦者以外の回線落ちがハイライトされます
* RMメニューにワンボタン配役設定機能が追加されます
