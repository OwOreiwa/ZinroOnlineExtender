人狼Onlineえくすてんだー
====
人狼Online（ https://zinro.net/ ）が使いにくいので自分で機能を追加する試みです  
現在1つの機能しかないですが他にも追加する予定です

Usage
----
* ブラウザアクションボタン押下で雑談系/身内/合言葉タグのついた村が非表示になります  
（精度は悪いので複数回実行してください）
* RMメニューにワンボタン配役設定機能が追加されます
